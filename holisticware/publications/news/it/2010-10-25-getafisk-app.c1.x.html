GETAFISK

Holistic VoIP Wizards strike again!! After our recent encounter with more than 300+ extensions to set up, dhcp, inboud routes and custom settings, we decided that it was time for us to build wrapper around Asterisk IP PBX (Elastix,Trixbox ...) so that we can automatically, with couple of mouse clicks, create all important configuration files needed for PBX to work. Plese stay tuned, as we will release beta soon :) http://getafisk.codeplex.com/
