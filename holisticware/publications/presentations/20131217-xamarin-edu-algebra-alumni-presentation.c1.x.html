Definitions

Xamarin concepts

  • native development for targeted platforms
  • cross platform support
    c# / f# as cross platform languages
  • sharing code accros platforms (NOTE: budiness logic code, domain code)
  • UI is written in c#, but API used is "native" (subnative, quasi native)
    AppKit/Cocoa in MonoMac/Xamarin.Mac, UIKit CococaTouch in Xamarin.iOS, Android in Xamarin.iOS, GTK# on Mac, Linux, Windows ...
  • open source mono, monodevelop and some tools/platforms (MonoMac, GTK#)
  • commercial packages (Xamarin.iOS, Xamarin.Android, Xamarin.Mac)
