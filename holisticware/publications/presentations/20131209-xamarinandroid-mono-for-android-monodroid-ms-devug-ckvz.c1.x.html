Summary / Conclusion

Xamarin.Android

    Xamarin.Android enables .net (C# / F#) to develop Android applications
  • fully native
  • using .net languages C# and F#
  • use .net BCLs and familiar frameworks
  • cons
    • learning curve for Android required (resources)
    • complexity (time and space)
      application (apk) size
      marshaling across boundaries (MCWs and ACWs)
      both are neglectable
  • pros
    • sharing code / reusing existing code base (libraries)
    • modern .net (LINQ, async/await, lambdas, anon methods)
