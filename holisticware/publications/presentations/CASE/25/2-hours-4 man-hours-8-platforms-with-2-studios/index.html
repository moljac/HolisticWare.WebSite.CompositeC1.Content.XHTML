Platforms + Architecture

      • Windows Phone

Platforms - Features

  • server
    • Display Slides (Images)

  • clients
    • desktop

Recommendations - Naming 1 UI Elements

UI elements domain neutral
  • Screen
    • Form - Windows Forms
    • Window - WPF, GTK, Mac (NSWindow), iOS (UIWindow)
    • Page - WPF, Silverlight (RIA, Windows Phone)
    • Activity - Android
  • Controls/Views/Widgets
    1st implemented platform rules?!?
    • WF or WPF - textBox1
      desktop app easier to develop or unit testing
    • iOS - UITextView - use textBox1
    • Android - EditText - use textBox1
    Events can be reused (not only events)
