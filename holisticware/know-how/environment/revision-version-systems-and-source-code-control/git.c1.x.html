Clone repository - Cloning a git repo means you are going to copy or download the source code 
to your local repository to work on.
Working on repository - You will be working on your local copy. Add or edit files 
and folders in your local copy.
Sync - Update your local repos with the server (download updated)
Commit - Confirm your works, update to local git
Push - update repository with local source to a writable repository
