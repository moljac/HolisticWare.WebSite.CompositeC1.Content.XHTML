Logon

Classic logon screen (with username and password) instead of Welcome Screen

The built-in Administrator account is hidden from Welcome Screen when a user account with Administrator privileges exists and enabled.

  • XP Home:
    login as Administrator only in Safe Mode
  • XP Professional, Vista Business:
    press CTRL +/ ALT +/ DEL twice to access classic logon screen

To turn off the Welcome screen and use the classic logon prompt from Windows 2000, which will require users to type a user account name.

  1. Start +/ Control Panel +/ User Accounts +/ Change The Way Users Log On or Off
  2. Use the Welcome screen = false (unchecked).
    Warning: This automatically disables Fast User Switching.

Registry entry: 

HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Winlogon\SpecialAccounts\UserList
  1. Backup key: File +/ Export
  2. Create a new DWORD Value named Administrator
  3. Edit or double-click Administrator and set Value to 1

Auto-logon

Note: security issue

Used for some testing projects and kiosk-like applications

Trim the file until it looks like following: 

Windows Registry Editor Version 5.00

[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Winlogon]
"AutoAdminLogon"="1"
"DefaultUserName"="frut"
"DefaultPassword"="frut.frut"


[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Winlogon\AutoLogonChecked]
  1. Start +/ netplwiz
    or
    Run +/ userpasswords2
  2. Users must enter a user name and password to use this computer +/ Uncheck
  3. Automatically Log On dialog box appears, enter the user name to automatically login.
  4. enter account password in the two fields where it's asked.
