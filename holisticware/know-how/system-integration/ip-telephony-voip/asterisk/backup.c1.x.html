Backup

Asterisk

OPERATION=cp -rf
# mv is not considered safe, cos asterisk may be writting to files use cp -a instead
# OPERATION=mv

DESTINATION=/tmp/asterisk-backup
mkdir $DESTINATION


$OPERATION \
	/etc/asterisk/* \
	$DESTINATION

$OPERATION \
	/var/spool/asterisk/* \
	$DESTINATION

$OPERATION \
	/var/lib/asterisk* \
	$DESTINATION

# if hardware present
$OPERATION \
	/etc/dahdi \
	$DESTINATION

FreePBX based Systems (Trixbox, Elastix, SwitchVox)

For FreePBX based systems, besides Asterisk script add:

  • dump/image of the Mysql database called "asterisk"
  • /usr/share/astguiclient/ADMIN_backup.pl --help
    the backup will be stored in /var/www/html/archive
