Conversion

Asterisk is actually able to convert a .wav file into any format for which it has a codec.
 
On the command line, simply run:
 
asterisk -x "file convert inputfile.wav outputfile.alaw"
 
...this would convert your wav file into an A-Law format file.
 
Alternatively, the program sox (available on Linux) is able to convert to/from loads of formats.
 
check out http://www.voip-info.org/wiki/view/Convert+WAV+audio+files+for+use+in+Asterisk for more methods and examples.
 
Hope this helps.
