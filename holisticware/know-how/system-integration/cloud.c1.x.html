Cloud

Application platforms

In late 2012, in order to reduce costs and boost it's knowledge, HolisticWare team started activities on recent hype terms and one of them was "Cloud" and "Cloud Technologies".

Being predominantly .net and mono developers team members started to investigate cloud application platforms for .net and mono. Primary properties that were investigated were

  • price (very important for HolisticWare and some of companies clients and partners)
  • installation and deployment (complexity, support).
    This factor was tested with deployment of Composite C1 CMS system.

Following cloud application platforms were tested:

  • Windows Azure http://www.windowsazure.com/
  • Appharbor http://appharbor.com/
  • Apprenda http://apprenda.com
  • Uhuru http://uhurusoftware.com/
  • Moncai http://moncai.com/
