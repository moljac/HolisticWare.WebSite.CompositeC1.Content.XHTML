Presentation on DevUG in Zagreb, Croatia on 2011-06-30

On 30th of June team held presentation about current state of the Mono for Android (earlier Monodroid), its current state, future and tips and tricks.

Presentation was intended as a intro for .net developers to Android platform, it's architecture, components, main development concepts, current state (versions) and development tools.

Second part of presentation was intended for .net programmers. Can they target Android platform, how, at what cost were some of the questions.

Presented was Mono for Android platform (Monodroid), some stuff from application team works on, VS solution with samples gathered from community, work with MonoDevelop etc...
