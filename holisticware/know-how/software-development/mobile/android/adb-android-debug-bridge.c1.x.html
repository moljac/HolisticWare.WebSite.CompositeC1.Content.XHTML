Wireless Deploying and Debugging (rooted device)

http://www.androidcentral.com/android-quick-app-adb-wireless
  1. ADB over WiFi from the device
    • Enable (port 5555) 
      setprop service.adb.tcp.port 5555
stop adbd
start adbd
    • Disable (adb will return to listening on USB) 
      setprop service.adb.tcp.port -1
stop adbd
start adbd
  2. adb over WiFi if usb connection is active
    • Enable (port 5555) 
      # usb device
adb -d tcpip 5555
# if only usb device is connected -d not necessary
adb tcpip 5555

adb connect 192.168.11.107:5559
					
      C:\Program Files\Android\android-sdk-windows\platform-tools>adb -d tcpip 5559
restarting in TCP mode port: 5559
    • Disable (adb will return to listening on USB) 
      adb usb
