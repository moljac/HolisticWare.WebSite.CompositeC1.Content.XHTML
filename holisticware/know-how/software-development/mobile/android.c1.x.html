Android

Definition

Android is an operating system primarily for mobile devices such as smartphones and tablet computers, but for desktop based computers to (AndroidX86).
It was developed by Android Inc., in 2005. and bought by Google. Further development is led by Google and Open Handset Alliance (80+ hardware, software and telecom companies).
Android is released under Apapche License (free)

    Android consist of:
  • kernel - Linux based
  • set of native (c/c++) libraries
  • application framework - java (modified)
    usage of Dalvik Virtual Machine (modified java VM) for dex code (Dalvil Executable) with just-in-time compilation (translated from java bytecode)
  • applications
    Development is done primarily in
  • java - primarily
  • Mono For Android (.net port for Android)
