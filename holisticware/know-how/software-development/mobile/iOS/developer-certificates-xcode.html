Visual Studio Integration

Date created : 2013-01-21
Date modified: 2013-01-21

To build apps for ad-hoc or deployment on two or more Macs, developer will need to export team signing certificate. Usually on the Mac from where it was created the first time.

  1. Locating certificates and exporting them on the original Mac box.
    1. Open up XCode
    2. Go to the organizer
    3. On the left side of the window, there are most likely have
      • LIBRARY,
      • TEAMS,
      • DEVICES.
    4. Click on TEAMS +/ TEAM_NAME
    5. In the detail window on the right side there are
      • company certificate and
      • developer (personal) certificate
    6. Company certificate will probably have little exclamation mark on it, that's ok. This will be fixed later.
    7. Refresh (Xcode will contact Apple's headquarters and refresh certificates)
    8. Export (either menu or bottom of the window) certificates Save profile and protect it with a password.
    9. There should be file name_given.developerprofile in the folder where exported (with xcode logo on top).
  2. Copy this developerprofile to other Mac and return repeat steps to locate certificate in Xcode, but this time import *.developerprofile under TEAMS.
