  • Resetting the iOS Simulator fixed the error for me. Although this will remove all of the Apps you have in Simulator, it fixes the problem without having to restart the machine. Reset your iOS Simulator by doing the following: 1) Go to the "iOS Simulator" menu, next to the Apple (?) logo on the far left of your main screen. 2) Select "Reset Content and Settings...". 3) Read the pop message and if you agree click "Reset" otherwise, click "Don't Reset". 
  • tell application "iPhone Simulator"

tell application "System Events"
	tell process "iPhone Simulator"
		tell menu bar 1
			tell menu bar item "iOs Simulator"
				tell menu "iOs Simulator"
					click menu item "Reset Content and Settings."
				end tell
			end tell
		end tell
		tell window 1
			click button "Reset"
		end tell
	end tell
end tell
