Troubleshooting

  • Symptoms:
    MonoTouch does not start (sometimes after update or license purchase).
  • Steps to fix (after each step test and proceed to next if it does no help):
    1. Check if ~/.recently-used is corrupted.
    2. Try removing ~/.config/MonoDevelop folder
    3. if above does not help try MonoTouch uninstall reinstall.
    4. remove file ~/.recently-used
