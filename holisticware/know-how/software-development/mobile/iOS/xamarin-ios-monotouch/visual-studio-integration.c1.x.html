Visual Studio Integration

Date created : 2012-02-21
Date modified: 2012-10-05

  1. Use plug-in/extension to enable Visual Studio to load MonoTouch projects.
    Here is plug-in developed by HolisticWare team MonoVersalProjectFlaves for Visual Studio 2010
    https://github.com/moljac/MonoVersalProjectFlavesForVisualStudio2010     HolisticWare plugin is based on Jamie Briant's plug-in https://github.com/jamiebriant/VsMono and adds ability to load other MonoDevelop projects like MonoMac and Windows Mobile projects (CE framework and SmartDevice) due to the fact that support for those projects is discontinued by Microsoft in Visual Studio version 2010 and above.
  2. Copy MonoTouch assemblies to some folder on Windows box.
    Source:
    • MacOSX
    • Extract MonoTouch package
    Destination:
    D:\SRC\Core_2011\Core.DLL_201_BL_mClnt_CrossPlatform_MobileAPI_Xamarin\lib
  3. Add destination folder to Visual Studio MonoTouch Project +/ Properties +/ Reference Paths.
