Windows Phone 7 WP7 Tools on Windows XP

WP7 tools are supported only on Vista and Windows 7 OS, but with small tweaks it is possible to develop (write apps and build), though testing is possible only on device (emulator cannot be installed and used)!

  • get iso.... from:
    • http://www.microsoft.com/downloads/en/details.aspx?FamilyID=04704ACF-A63A-4F97-952C-8B51B34B00CE
    • link to ISO file http://go.microsoft.com/fwlink/?LinkId=201927
    • update 2011-01 http://www.microsoft.com/downloads/en/details.aspx?FamilyID=49B9D0C5-6597-4313-912A-F0CCA9C7D277
  • Unpack iso/mount it...
  • cd to WCU\
  • install:
    • Silverlight 4
    • Blend
    • use script (create batch script and start it in WCU folder):
      • echo off
