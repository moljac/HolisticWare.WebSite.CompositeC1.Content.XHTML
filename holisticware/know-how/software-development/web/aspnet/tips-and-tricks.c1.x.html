Visual Studio temp files

Visual Studio Exteral tool (ASP.net temp files) 

Title: 				ASP.net Temporary Files Location
Commnad:			explorer.exe
Arguments:			/e,/root,"%USERPROFILE%\AppData\Local\Temp\Temporary ASP.NET Files"
Initial directory:	$(ProjectDir)

Command (put in batch or paste into (run (ctrl +/ r) TextBox)

Clean - all - DistClean

Besides Project/Solution clean, clean temporary files too, in: 

	%USERPROFILE%\AppData\Local\Temp\Temporary ASP.NET Files\
	%USERPROFILE%\AppData\Local\Temp\Temporary ASP.NET Files\*
	%USERPROFILE%\AppData\Local\Temp\Temporary ASP.NET Files\VirtualDirectoryName
