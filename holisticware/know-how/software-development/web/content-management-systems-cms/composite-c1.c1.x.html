Composite C1

In 2011-02 requirement to do a web site preferably CMS for one company. From own point of view system should have following features:

  • lightweight (no database if possible)
  • ASP.net as technology (for extensibility)

After a week of searching the result was Composite C1 CMS system, excellent CMS system that can run w/o database with plain XML based web site...

This is HolisticWare wiki for knowledge gathered about Composite C1 and stuff around it...

Documentation (http://docs.composite.net/C1.aspx)
Discussions (http://compositec1.codeplex.com/discussions)

Composite C1 is CMS system written in .net 4.0 which utilizes latest Microsoft technology (ASP.net 3.5, LINQ, MVC Razor, ...)
