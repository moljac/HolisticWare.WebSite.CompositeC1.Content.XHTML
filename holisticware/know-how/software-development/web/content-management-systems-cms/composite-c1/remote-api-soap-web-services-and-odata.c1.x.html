validate a user using

Composite.C1Console.Security.UserValidationFacade.FormValidateUser(username, password);

 

This call will set a cookie on the current request and the 
validity of this cookie can be verified by this call

	Composite.C1Console.Security.UserValidationFacade.IsLoggedIn()

To get the username for a user that have been validated earlier use

	Composite.C1Console.Security.UserValidationFacade.GetUsername()

 

OData/SOAP client behaviour:


1.       Given user name and password call a service which call FormValidateUser
2.       On the client, if this call returns true, keep authentication cookie and send 
		 it along all further requests
a.       On server, for data functions use IsLoggedIn to verify you have an authenticated user
