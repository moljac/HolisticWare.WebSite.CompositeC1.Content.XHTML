Navigation Distributed - find level automagically

When using Composite.Navigation.Distributed user has to specify level (3 in this case) 

	<f:function <br/>
		name="Composite.Navigation.Distributed" <br/>
		xmlns:f="http://www.composite.net/ns/function/1.0"<br/><br/>
		>
	  <f:param name="Level" value="3" />
	  <f:param name="Childs" value="True" />
	  <f:param name="Expand" value="True" />
	</f:function>

How can Level be obtained automagically in XML in call of function above?
Expression instead of constant 3!
