MySql Data Store

Prerequisites

  • MySql Connector for .net
    MySql Connector at http://www.mysql.com/downloads/connector/
    MySql Connector for .net download at http://www.mysql.com/downloads/connector/net/
  • DbLinq support for Linq
    DbLinq site at http://code.google.com/p/dblinq2007/ DbLinq svn repo
    DbLinq svn repo at http://dblinq2007.googlecode.com/svn/trunk

Install

  • install MySql Connector
  • Checkout source of DbLinq from svn
  • CompositeC1 Solution + Add existing projects
    • Add Existing Project + Browse to: DbLinq.MySql
    • Add Existing Project + Browse to: DbLinq
    • CompositeC1 Solution + Composite Project + Add Reference + Assemblies + Extensions + MySql.Data
