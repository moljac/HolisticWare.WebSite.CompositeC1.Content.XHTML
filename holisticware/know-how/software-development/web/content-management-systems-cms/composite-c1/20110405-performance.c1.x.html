Performance

HolisticWare team considers performance as a complex problem consisting of

  • page speed which is influenced through its
    • size (size of components, html, pictures, javascript, style-css)
    • number of components - this defines number of requests
    • order of components - some requests might block other, making pages appear slow
    • algorithms - caching, delayed/deferred downloads
  • content and information, presented to others (search engines)
  • user experience UX (although this is influenced through two above too)
