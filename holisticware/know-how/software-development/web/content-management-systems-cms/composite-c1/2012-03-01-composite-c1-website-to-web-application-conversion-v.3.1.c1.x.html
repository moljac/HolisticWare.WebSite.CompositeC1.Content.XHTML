Steps:

  1. Add Web Application Project to the Composite C1 solution
  2. Select all files except Bin\ folder in Website folder, copy them and paste into Web Application (WebApplication1 - most likely) folder
  3. In Web application folder create lib (or similar folder) and copy referenced assemblies from WebSite\Bin\* into this folder.
  4. Add references from WebApplication1\lib*.dll folder!
  5. Optional: Rebuild
  6. Context Menu (Right Click) on WebApplication1 +/ Convert to Web Application
  7. Errors like:
