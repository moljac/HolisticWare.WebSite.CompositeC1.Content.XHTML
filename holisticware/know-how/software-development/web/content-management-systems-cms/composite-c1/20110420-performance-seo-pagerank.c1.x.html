As a result of work to force CMS content editors and developers to write meta tags important for SEO for each content page a XSLT function was developed.

  • Function
  • Usage - this XSLT function is used as
    • placeholder for several html meta-tags used for SEO and
    • SEO enforcer (actually forces developer/content editor to add SEO data/meta-tags, because all mandatory information has no default parameters, so user has to enter it during editing.
      • Function code:
      • Function parameters:
      • Function's output in html: 
        <html xmlns="http://www.w3.org/1999/xhtml">
<head>
	<meta http-equiv="content-type" content="text/html; charset=iso-8859-1" />
	<title>HolisticWare - Title</title>
</head>
<body>
	<f:function name="HolisticWare.Optimizations.SearchEngine.MetaTagsKeywords" xmlns:f="http://www.composite.net/ns/function/1.0">
	  <f:param name="KeywordsAdditional" value="title" />
	  <f:param name="AuthorsAdditional" value="mcvjetko" />
	  <f:param name="Date" value="2011-04-18" />
	  <f:param name="Language" value="en" />
	</f:function>
	<h1>HolisticWare - Title</h1>
