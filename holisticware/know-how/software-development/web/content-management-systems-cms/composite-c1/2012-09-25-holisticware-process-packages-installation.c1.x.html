HolisticWare Process Composite C1 package installation

Steps

  1. manual
    http://users.composite.net/Superuser/Packages
    http://users.composite.net/Superuser/Packages/InstallPackage
    http://c1packages.codeplex.com/
  2. batch install
    1. Check-out out packages from codeplex
      http://docs.composite.net/Packages
    2. wrote batch that collects all zips in one folder AutoInstallPackages
    3. copy folder to ~/App_Data/Composite/
    4. start website
      http://msdn.microsoft.com/en-us/library/xee0c8y7.aspx 
      					devenv Composite.sln /Rebuild
					devenv Composite.sln /Run
    5. script currently developed 
      :: holisticware-proces-packages-install-batch.bat
