Starter Site Features:

  • Desktop and mobile friendly. Based on cssgrid.net 12 column grid.
  • 5 basic templates: Frontpage, full width, 2 and 3 columns.
  • Top menu
  • Quick links like sitemap and other languages
  • Left menu, based on Composite.Navigation.Distributed package
  • Language switcher, based on Composite.Navigation.LanguageSwitcher package
  • Sitemap, based on Composite.Navigation.LevelSitemap package
  • Clean and lightweight markup and CSS - you can work with
  • Easy skinning
  • HTML meta tags done
