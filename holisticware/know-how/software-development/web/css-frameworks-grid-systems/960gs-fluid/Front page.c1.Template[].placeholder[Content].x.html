Composite C1 Starter Site

Hi, and welcome to your Composite C1 starter site. This site is a set of minimalistic templates for starting a new site. You can use this site "as is" with light re-skinning, you can extend it with C1 Packages (blogs, extranet, event calendars etc.) and you can build a highly customized website on top of it.

If you are new to Composite C1:

Wish to edit this website? You can launch the C1 Console in your web browser and try to edit pages, upload images etc. with Composite C1. Read more in "How to edit this website" .

  • Start the C1 Console - this is where you work with your content etc. 
  • Getting started with Composite C1 - how to edit pages etc.
  • Videos for new users - how you log in, find and edit pages etc.
  • Installing packages - how you add features like blog, extranet, image galleries etc.

If you are a web developer:
