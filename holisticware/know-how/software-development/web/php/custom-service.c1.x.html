Sending hehehe string! http://www.ubique.hr/Customers/test_php_service.php?text=hehehe

Code for test_php_service.php: 

<?php

/* require the text as the parameter */
if(isset($_GET['text'])) {
{
		header('Content-type: text/xml');
		echo '<text>'.$_GET['text'].'</text>';
	}

}
?>
