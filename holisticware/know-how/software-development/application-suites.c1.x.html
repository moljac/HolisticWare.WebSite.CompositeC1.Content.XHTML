Application Suites

Intro

Application suites can be defined as:

  • Complementary App Suites
    group of complementary applications packaged to solve different, but common problems on some platform
  • Multiplatform Appsuites
    group of similar applications that solve the same function on several platforms

Complementary App Suites

A group of programs that are packaged to solve common problems in a certain problem domain. So if one defines problem domains as graphics, mathemtics, science, I[C]T, so there are suites for graphics, mathematics and other applications.

The most popular are "office suites", also known as "productivity suites," defined as a set of basic business programs designed with a uniform user interface and common functions such as spell checking. The primary programs are word processing, spreadsheet, presentation graphics, database and e-mail, although each suite has its own mix, and a variety of other programs and utilities may also be included.
Some of the programs may be for sale as individual products.
