Solution Loading

  1. P: Projects grayed out (not built in active configuration)
    S: Change active configuration type (let say from "Debug" to "Debug|Mixed Platforms")
  2. P: Projects red (Load failed)
    S: No solution - project types not supported (Windows Phone, WPF) or plug-in
