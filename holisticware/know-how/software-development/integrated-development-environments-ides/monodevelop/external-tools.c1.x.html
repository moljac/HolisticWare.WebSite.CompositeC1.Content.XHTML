Sample

MonoDevelop [External] Tools are like Visual Studio External Tools.
Integration of UI WYSIWYG Designer tools for Android:
DroidDraw:

Eclipse:

Only Difference are files where settings are stored: 

%USERPROFILE%\AppData\Roaming\MonoDevelop-2.8\Config\MonoDevelop-tools.xml

Resulting in: 

<?xml version="1.0" encoding="utf-8"?>
<Tools version="2.0">
	<ExternalTool 
		menuCommand="Mono4Android DroidDraw" 
		command="c:\bin\android\droiddraw\droiddraw.exe" 
		arguments="${FilePath}" 
		initialDirectory="${FileDir}" 
		promptForArguments="False" 
		useOutputPad="True" 
		saveCurrentFile="True" 
	/>
	<ExternalTool 
		menuCommand="Mono4Android Eclipse" 
		command="c:\bin\eclipse\eclipse.exe" 
		arguments="--launcher.openFile ${FilePath}" 
		initialDirectory="${FileDir}" 
		promptForArguments="False" 
		useOutputPad="True" 
		saveCurrentFile="False" 
	/>
</Tools>
