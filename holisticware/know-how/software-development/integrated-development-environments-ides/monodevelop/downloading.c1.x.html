Downloading various Versions

Downloading in the format:

  • Windows
    http://download.xamarin.com/monodevelop/Windows/MonoDevelop-${VERSION}.msi
  • Mac OSX
    http://download.xamarin.com/monodevelop/Mac/MonoDevelop-${VERSION}.dmg

Replace ${VERSION} with the particular version wanted.
