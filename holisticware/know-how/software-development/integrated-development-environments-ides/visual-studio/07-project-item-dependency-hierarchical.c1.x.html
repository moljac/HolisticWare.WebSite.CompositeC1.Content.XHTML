Project Items dependency - hierarchical

In project file (*.csproj) there is c# code file project item denoted with action Compile (Build): 

		<Compile Include="Resources\Layout\ActivityWithLayoutMain.axml.cs" />

Changing cs item to 

		<Compile Include="Resources\Layout\ActivityWithLayoutMain.axml.cs" >
			<DependentUpon>ActivityWithLayoutMain.axml</DependentUpon>
		</Compile>
