Debugging and Deploying

  1. Problem:
    Hitting F5 causes deployment of other projects
    (for example Opening Device List in Mono For Android to choose deployment device: emulator or physical device).
    • triggered Deploy Action
    • Multiple Startup Options ?!?!
  2. Solution:
    • Disable Deployment in Configuration Manager
      Configuration Manager/Build and Deploy checkboxes
      Uncheck Deploy
      Not feasible, because it must be done for each change of Startup Project! Change both Startup Project and Enable Deployment for Startup Project and disable Deployment for previous Startup Project
      Switching Active Configurations
    • Change Options:
      Tools/Options/Projects and Solutions/Build And Run/
      "Only build startup projects and dependencies on Run" = true
      Check "Only build startup projects and dependencies on Run".
      Applies to Deploy too!
      Use Mixed Platforms configuration - only refrenced by Startup Project will be built, deployed and debugged when pressing F5!
