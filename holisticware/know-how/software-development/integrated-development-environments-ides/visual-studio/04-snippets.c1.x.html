Snippets

Recommendations - Naming

Recommended naming for Visual Studio snippets on HolisticWare projects: 

	[prefix]_[technology]_[techonlogy_subclass]_[loation]_description.snippet
  • prefix := user | company
  • technology:
    • WP - windows phone
    • SL - Sliverlight
    • WPF
    • WF - WIndows Forms
    • mtch - MonoTouch
    • m4a, mdrd - Mono for Android / MonoDroid
  • location (inserting position):
    • F - file level
    • N - namespace level
    • C - class level
    • M - method (procedure, function) level
    • P - property level

Publishing

Create .vscontent file for a .vsi that contains a code snippet. The following files would be included in the .vsi file for this content. 

<VSContent xmlns="http://schemas.microsoft.com/developer/vscontent/2005">
    <Content>
        <FileName>MyCodeSnippet.snippet</FileName>
        <DisplayName>Sample Code Snippet</DisplayName>
        <Description>A code snippet created for this example</Description>
        <FileContentType>Code Snippet</FileContentType>
        <ContentVersion>1.0</ContentVersion>
        <Attributes>
            <Attribute name="lang" value="vb"/>
        </Attributes>
    </Content>
</VSContent>
