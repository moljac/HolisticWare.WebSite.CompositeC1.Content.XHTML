Html Encode Macros

To HTML encode selected text (code snippets from source files like *.cs, *.xml,...) in html documents.
Select text and execute macro (keyboard shortcut).

MyMacros.HtmlEncode.HtmlEncode 

Imports System
Imports EnvDTE
Imports EnvDTE80
Imports EnvDTE90
Imports EnvDTE90a
Imports EnvDTE100
Imports System.Diagnostics

Public Module HtmlEncode

	Sub HtmlEncode()
		DTE.ActiveDocument.Selection.Text = _
		System.Web.HttpUtility.HtmlEncode(DTE.ActiveDocument.Selection.Text)
	End Sub

End Module

Keyboard Shortcut:

Title convert to SEO friendly filename

Convert Title to filename (replace space with - and convert to lowercase)

MyMacros.HtmlFilenameSEOFriendly.HtmlFilenameSEOFriendly 

Imports System
Imports EnvDTE
Imports EnvDTE80
Imports EnvDTE90
Imports EnvDTE90a
Imports EnvDTE100
Imports System.Diagnostics

Public Module HtmlFilenameSEOFriendly

	Sub HtmlFilenameSEOFriendly()
		Dim txt As String
		txt = DTE.ActiveDocument.Selection.Text()
		txt = txt.Replace(" ", "-")
		txt = txt.ToLower()
		DTE.ActiveDocument.Selection.Text = txt
	End Sub

End Module
