Solution

Changing Build Action from Page to None or Content 

	<Page Include="VisualStudio\2010\templates\ProjectTemplates\MultiTargetSolution\Company.Project.EXE_811_PL_mClnt_WP7_SL4_netfx35\App.xaml">
	  <Generator>MSBuild:Compile</Generator>
	  <SubType>Designer</SubType>
	</Page>

Changes item in project to: 

	<None Include="VisualStudio\2010\templates\ProjectTemplates\MultiTargetSolution\Company.Project.EXE_811_PL_mClnt_WP7_SL4_netfx35\App.xaml">
	  <Generator>MSBuild:Compile</Generator>
	  <SubType>Designer</SubType>
	</None>

This influences Core project (content placeholder, but not the projects deep in folder structure used as solution template).
